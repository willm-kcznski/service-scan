# 🚀 em-midori-forge

> API gateway for microservices

## ✨ Features

- Fast compilation (sub-second builds)
- Hot reload included
- Plugin system
- Zero config start

## 📦 Installation

```bash
npm install -g em-midori-forge
em-midori-forge init my-project
```
